Blinkit Sales Dashboard

This project analyzes Blinkit sales data using an interactive dashboard to identify
key business insights across product categories, outlet types, sizes, and locations.

Contents:
- Dashboard image showing sales KPIs and trends
- Dataset used for analysis
- Executive summary highlighting key insights

Key Insights:
- Total sales of $1M with an average rating of 3.9
- Supermarket Type 1 and high-size outlets drive maximum revenue
- Tier 3 locations outperform Tier 1 and Tier 2
- Regular-fat products dominate overall sales

Tools Used:
- Power BI / Excel 
- Data visualization and analysis techniques
